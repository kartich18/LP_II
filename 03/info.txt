Technical Details - Restaurant Menu Website

1. File Structure:
- index.html: Main HTML structure
- styles.css: CSS styling

2. Technologies Used:
- HTML5
- CSS3
- Responsive Design
- CSS Grid & Flexbox

3. Sections:
a) Header:
   - Restaurant name
   - Tagline
   - Background image

b) Welcome Section:
   - Restaurant introduction
   - Chef's message
   - Chef's image

c) Popular Dishes:
   - Ordered list
   - Top 5 dishes

d) Cuisines Section:
   - Grid layout
   - Image cards
   - Hover effects

e) Menu Section:
   - Grid layout
   - Dish cards with images
   - Pricing
   - Descriptions

4. Styling Features:
- CSS Grid for layouts
- Flexbox for alignment
- Responsive breakpoints
- Hover animations
- Box shadows

5. Image Handling:
- Responsive images
- Object-fit cover
- Border radius
- Overlay effects
- Placeholder images from picsum.photos

7. Design Elements:
